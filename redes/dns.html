Domain Name System


“Servidor de Domínios”


O que é domínio?

O domínio é o nome do site na web e serve para facilitar a navegação do usuário, que acaba não precisando lembrar o IP de cada site.

  • O DNS realiza a tradução do nome de um domínio para o endereço de IP. Existem vários servidores DNS no mundo e é fundamental para a nossa web o funcionamento deles.
  • Cliente-Servidor
  • O cliente envia uma requisição ao Servidor DNS
  • O servidor DNS retorna o endereço ip por meio do qual o cliente envia requisições ao servidor do domínio que deseja acessar.
  • Raiz do domínio
    • .com (comercial)
    • .br (país)
    • .org (organização)
    • .net (rede)
    • outros
