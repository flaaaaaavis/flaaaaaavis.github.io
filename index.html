Open Graph

Open Graph é uma tecnologia que te permite ter controle sobre como sua página web será compartilhada nas redes sociais.



As tags Open Graph são o seu segredo para uma boa CTR nas redes sociais

Open Graph é um protocolo usado por redes sociais para definir como uma URL deve ser exibida quando compartilhada nas redes sociais. Trechos especiais de código chamados tags OG especificam como elementos como título, descrição e imagem de visualização devem ser mostrados nos feeds sociais.
Uma visualização bem otimizada e direcionada de seu URL ajudará a melhorar significativamente o CTR de sua postagem nas redes sociais.
