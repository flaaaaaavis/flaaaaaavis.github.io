O que é SQL

SQL (Structured Query Language) é uma linguagem usada para executar comando em bancos de dados relacionais, isto é, baseado em tabelas. Apesar de não ser o único meio utilizado para este fim, certamente, é o mais popular. Em resumo, podemos dizer que essa linguagem de programação foi criada para que os programadores pudessem simultaneamente, acessar, consultar e modificar os dados estruturais de uma empresa ou organização.
