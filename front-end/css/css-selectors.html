CSS Selectors

Cascade

"When two rules apply that have equal specificity, the one that comes last in the CSS is the one that will be used."

Specificity

Every CSS selector has its place in the specificity hierarchy.
There are four categories which define the specificity level of a selector:

  1. Inline styles
  2. IDs
  3. Classes, pseudo-classes, attribute selectors
  4. Elements and pseudo-elements

Inheritance

"Some CSS property values set on parent elements are inherited by their child elements, and some aren't."

Seletor Atributo

Permite a seleção de um elemento com base na presença de um atributo, ou em várias correspondências diferentes com o valor do atributo.


Pseudo-classe

Uma pseudo-classe é um seletor que seleciona elementos que estão em um estado específico, por exemplo, eles são o primeiro elemento de seu tipo ou estão sendo passados ​​pelo ponteiro do mouse. Eles tendem a agir como se você tivesse aplicado uma classe a alguma parte do seu documento, geralmente ajudando a reduzir o excesso de classes em sua marcação e fornecendo um código mais flexível e sustentável.

Todas as pseudoclasses se comportam da mesma maneira. Eles têm como alvo uma parte do seu documento que está em um determinado estado, comportando-se como se você tivesse adicionado uma classe ao seu HTML.


Pseudo-elemento

Agem como se você tivesse adicionado um novo elemento HTML inteiro à marcação, em vez de aplicar uma classe aos elementos existentes. Por exemplo, se você quiser selecionar a primeira linha de um parágrafo, poderá envolvê-la em um elemento span e usar um seletor de elemento; entretanto, isso falharia se o número de palavras que você empacotou fosse maior ou menor do que a largura do elemento pai. Como tendemos a não saber quantas palavras caberão em uma linha - já que isso vai mudar se a largura da tela ou o tamanho da fonte mudar - é impossível fazer isso de forma robusta adicionando HTML.


Combinators

Combina outros seletores de uma forma que forneça uma relação útil entre eles e a localização do conteúdo no documento.
