Imagens no HTML

As imagens são inseridas em nosso documento HTML usando a tag <img>. A tag de imagem é uma tag de fechamento automático e não exige uma tag de fechamento correspondente.

Anatomia básica da tag <img>

  • src - a URL da imagem. Este atributo é obrigatório para o elemento <img>.
  • alt - este atributo define a descrição da imagem. Os usuários verão esse texto exibido se a URL da imagem estiver incorreta, se a imagem não estiver em um dos formatos suportados ou se a imagem ainda não tiver sido baixada.
  • height - a altura da imagem em pixels
  • width - a largura da imagem em pixels
