Tags da estrutura básica de páginas HTML

<!DOCTYPE html>

Informa ao navegador qual versão do HTML será usada neste documento. Neste caso, estamos declarando o HTML5.

Doctype é especial no sentido de que ele não é considerado uma tag de verdade, mas sim uma declaração. O que você realmente precisa entender sobre esta tag/declaração é que ela deve estar presente, e deve estar especificamente na primeira linha do seu arquivo HTML.

<html>

É a tag que representa o elemento raiz (elemento de nível superior) de um documento HTML, por isso ela também é chamada de elemento raiz. Todos os outros elementos devem ser descendentes deste elemento.

<head>

O head é a tag que contém informações descritivas sobre a sua página, como o título (title) das páginas. Também é aqui onde você insere coisas como meta e links de estilo.

<title>

O title é a tag que define o título do documento que é exibido na barra de título do navegador ou na aba da página. Esta tag pode conter apenas texto e qualquer outra tag colocada dentro dela é ignorada.

<body>

O body representa o conteúdo de um documento HTML. Pode haver apenas um elemento body em um documento HTML. É aqui onde ficam as coisas que queremos que as pessoas vejam.

Tags HTML

Títulos

As tags HTML <h1> - <h6> representam seis níveis de título de seção. O <h1> é o nível mais elevado, e o <h6> é o mais baixo.

Parágrafo

A tag HTML <p> representa um parágrafo de texto.

Div

O elemento HTML <div> é um recipiente genérico para fluxo de conteúdo. Ele é usado principalmente para agrupar elementos para estilização e posicionamento.

A <div> é um elemento de nível de bloco. Os navegadores normalmente exibem o elemento de nível de bloco colocando uma nova linha antes e depois do elemento.

Span

O elemento HTML <span> é um container genérico para o fluxo de conteúdo. Ele é usado principalmente para fins de estilo.

O <span> é um elemento em linha. Um elemento em linha ocupa apenas o espaço delimitado pelas tags que o definem. Basicamente isso significa que você pode colocá-lo em volta de uma palavra ou algo semelhante e ele não vai adicionar novas linhas acima e abaixo do span.
